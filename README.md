# 📱 SMS Receiver — Números Virtuales Gratuitos

Proyecto Python para consultar y monitorear mensajes SMS recibidos en números virtuales públicos gratuitos, usando la librería `receive-sms`.

---

## ⚡ Inicio rápido (GitHub Codespaces)

1. Abre el repositorio en GitHub
2. Click en **Code → Codespaces → Create codespace on main**
3. Espera que el entorno se configure automáticamente
4. En la terminal ejecuta:

```bash
python main.py
```

---

## 🖥️ Instalación local

```bash
git clone https://github.com/TU_USUARIO/sms-receiver.git
cd sms-receiver
pip install -r requirements.txt
python main.py
```

---

## 🧩 Funcionalidades

| Opción | Descripción |
|--------|-------------|
| `1` | Ver mensajes de números predefinidos |
| `2` | Consultar cualquier número personalizado |
| `3` | Monitoreo continuo con intervalo configurable |

---

## ⚠️ Advertencia importante

> Los números virtuales usados son **públicos** — cualquier persona puede ver los mensajes.
> No uses este servicio para recibir información sensible o confidencial.
> Ideal para: pruebas, verificaciones de apps, notificaciones no críticas.
